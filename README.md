Margin, Padding e Borger
As propriedades de margem CSS são usadas para criar espaço ao redor dos elementos, fora de quaisquer bordas definidas.

As margens definem a distância entre a borda de um elemento e os elementos circundantes.

Com CSS, você tem controle total sobre as margens. O CSS possui propriedades para definir a margem de cada lado individual de um elemento (superior, direita, inferior e esquerda), além de uma propriedade abreviada para definir todas as margens em uma única declaração.

Como usar
A propriedade margin pode ser especificada usando um, dois, três ou quatro valores. Cada valor deve ser um [<length>], uma [<percentage>], ou a palavra-chave [auto]. Cada valor pode ser positivo, zero ou negativo.

Exemplo:
div {
margin: 20px;
}

Padding
O padding é o espaço interno de um elemento, ou seja, a distância entre o conteúdo (texto, imagem, etc.) e a borda.

Como usar
Você usa o padding dentro do CSS para criar espaço dentro do elemento

Exemplo:
div {
padding: 20px;
}

Border
A border é a borda de um elemento, ou seja, o contorno que fica ao redor dele.
Ela serve para destacar, separar e organizar os elementos na página.

Como usar
O border é usado dentro do CSS para colocar uma borda em um elemento (como uma div, botão, texto, etc.).
Exemplo:

div {
border: 2px solid black;
}
